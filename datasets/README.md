# Datasets

Thư mục này dùng để chứa dữ liệu train, validate, test và video chạy thử.

- `vehicle_detection/` - Dataset nhận diện xe/người.
- `traffic_sign/` - Dataset nhận diện biển báo giao thông.
- `lane_segmentation/` - Dataset phân đoạn làn đường.
- `vietnam_test_videos/` - Video test đường Việt Nam.

Không commit dữ liệu lớn lên Git. Chỉ giữ README và file `.gitkeep` để giữ cấu trúc thư mục.

Sau khi clone project, tải dataset từ link của team rồi giải nén đúng vào các thư mục trên.
